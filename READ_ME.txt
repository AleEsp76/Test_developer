## Preparacion:

Algunos programas que se necesitan para compilar este programa son:

-Python: para la instalacion de python entraremos a la pagina oficial de descargas de python https://www.python.org/downloads/windows/
y descargar el instalador de la version mas reciente para el sistema operativo deseado. 

-Visual Studio Code: para descargar este programa iremos a la pagina oficial de descarga de visual studio code y descargamos la version 
del programa para el sistema operativo deseado.

-requests: Para instalar requests es necesario tener python, ya que necesitaremos el instalador de paquetes pip(el cual se incluye en la 
instalación de python) y para poder conseguir requests entraremos al cmd de la computadora y escribiremos "pip install requests"

-pandas: Para instalar pandas es necesario tener python, ya que necesitaremos el instalador de paquetes pip(el cual se incluye en la 
instalación de python) y para poder conseguir requests entraremos al cmd de la computadora y escribiremos "pip install pandas"

## Manual de Uso

Este proyecto cumple con la funcion de obtener informacion de una API a partir de un URL modificado y de un archivo cvs externo y 
guardarla en un archivo .cvs, asi como crear un archivo .txt cuando detecta algun error y anotar el error. Para poder ingresar a 
la API se deben de colocar correctamente los permisos de entrada si no la API no regresara ninguna informacion.
