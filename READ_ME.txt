## Program Requirements:

-Python 3.7: For the installation of python we will enter the official python download page https://www.python.org/downloads/windows//

-Have pip installed.

-Run the commmand "make install_requirements" in the terminal, this is to install all the necessary dependencies. 

## Manual de Uso

This project fulfills the function of obtaining information from an API from a modified URL and an external cvs file and
save it in a .cvs file, as well as create a .txt file when it detects an error and write down the error. In order to enter
The API must correctly set the input permissions, otherwise the API will not return any information.
